# repo2slides
Repo2Slides是一个命令行工具，可将GitHub仓库自动转换为学术风格的PPT大纲，面向学生、研究者和开发者。核心功能：自动分析仓库结构，识别代码目录、实验脚本、结果文件；解析 README，提取背景、方法、实验、结果；检测 ML 框架（PyTorch、TensorFlow 等）；分析 metrics 文件，提取关键指标；生成 8-12 页的学术风格 PPT 大纲。 技术特点：支持 LLM 增强模式和纯启发式离线模式；轻量级代码解析，只提取结构信息；自动识别中英文章节；支持中英文两种 PPT 风格；输出 Marp/reveal.js 兼容的 Markdown，也可导出为 PPTX。 使用方式：一行命令即可生成 PPT 大纲，适合需要快速准备项目汇报的场景，尤其适合 ML/研究类项目。
